# Oracle JET Data visualization based on Zeppelin syntax.

Oracle JET Data visualization based on Zeppelin syntax.

https://www.oracle.com/webfolder/technetwork/jet/index.html
https://classic.yarnpkg.com/en/
https://jestjs.io/

## Installation

macOS

```bash
npm install -g yarn && yarn install
```

## Usage

```bash
yarn build
yarn serve
```

## Test
```bash
yarn test
```
